# Financial Design System

## Visual Direction

This is a personal finance application.

The UI should be:
- professional
- trustworthy
- minimal
- data focused

## Rules

- Always use shadcn components.
- Never create custom UI primitives if shadcn has an equivalent.
- Never create new colors.
- Never introduce new spacing patterns.
- Reuse existing components before creating new ones.
- Keep dashboards dense and information-oriented.

## Components

Prefer:
- Card
- Table
- DataTable
- Form
- Sheet
- Dialog
- Tabs
- Badge
